# Korvenix Paperclip Skills

Agent skills for Korvenix. Forked from [paperclipai/companies](https://github.com/paperclipai/companies) (aeon-intelligence).

## Skills

- pr-review — code review for pull requests
- github-monitor — monitor watched repos for activity
- issue-triage — triage and categorize GitHub issues
- changelog — generate changelogs from commits and PRs
- code-health — analyze codebase health metrics
- feature — implement features on branches
- build-skill — create new skills
- search-skill — discover community-built skills
